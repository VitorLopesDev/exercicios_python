# Exercícios Iniciais em Python

Este repositório contém uma coleção de exercícios básicos desenvolvidos para praticar lógica de programação, manipulação de strings, operações matemáticas e condicionais em Python.

# Descrição dos Exercícios

### 1 - Concatenando Dados

Recebe dois dados do usuário e concatena em uma única string.
Conceitos trabalhados: entrada de dados, strings, concatenação.

### 2 - Repetindo Textos

Recebe uma string e um número inteiro, repetindo o texto a quantidade de vezes informada.
Conceitos trabalhados: laços implícitos de repetição, strings, tipos numéricos.

### 3 - Operações Matemáticas Simples

Solicita dois números e realiza: soma, subtração, multiplicação e divisão.
Conceitos trabalhados: operadores matemáticos, entrada de dados, tipos numéricos.

### 4 - Verificando Números Pares e Ímpares

Recebe um número inteiro e verifica se é par ou ímpar usando condicionais.
Conceitos trabalhados: if, else, operador %.

### 5 - Calculando Média de Notas

Lê três notas e calcula a média delas.
Conceitos trabalhados: variáveis, operações aritméticas, entrada de dados.

### 6 - Verificando Palíndromos

Verifica se uma palavra é um palíndromo comparando a versão normal e a invertida.
Conceitos trabalhados: manipulação de strings, slicing.
